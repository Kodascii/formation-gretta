# (Conda) Jupyter

## Setup
<u-b>1. INSTALL</u-b>
```sh
$ conda install jupyterlab
```

<u-b>2. INSTALL NOTEBOOK</u-b>
```sh
$ conda install notebook
```

## Extras
<u-b>RUN NOTEBOOK</u-b>
```sh
$ jupyter notebook
```

<u-b>RUN JUPYTERLAB</u-b>
```sh
$ jupyter lab
```
